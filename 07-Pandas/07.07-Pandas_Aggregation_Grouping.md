# Pandas: Aggregation & Grouping

## Overview
Aggregation and grouping are among the most powerful concepts in Data Analysis. In Pandas, this process is built around the **"Split-Apply-Combine"** methodology. You split the data into groups based on some criteria, apply a function (like sum, mean, or count) to each group independently, and then combine the results back into a new DataFrame.

This technique is essential for summarizing large datasets and answering business questions like "What is the total revenue per region?" or "What is the average salary by department?"

---

## 1. The `groupby()` Mechanism
The `groupby()` function in Pandas is used to group large amounts of data and compute operations on these groups. 

By itself, `df.groupby('Column')` just creates a `DataFrameGroupBy` object in memory. It doesn't compute anything until you chain an aggregate function to it.

```python
import pandas as pd

# Sample Dataset
data = {
    'Department': ['Sales', 'HR', 'Sales', 'IT', 'HR', 'IT'],
    'Employee': ['Alice', 'Bob', 'Charlie', 'David', 'Eve', 'Frank'],
    'Salary': [60000, 50000, 70000, 80000, 55000, 85000],
    'Experience_Years': [3, 2, 5, 8, 4, 10]
}
df = pd.DataFrame(data)

# Creating a GroupBy object
grouped_df = df.groupby('Department')
print(grouped_df) 
# Output: <pandas.core.groupby.generic.DataFrameGroupBy object at ...>
```

---

## 2. Basic Aggregation Functions
Once data is grouped, you can apply summary statistics to the numerical columns.

```python
# Calculate the mean salary for each department
avg_salary = df.groupby('Department')['Salary'].mean()
print(avg_salary)

# Calculate the total salary payout per department
total_salary = df.groupby('Department')['Salary'].sum()

# Count the number of employees in each department
emp_count = df.groupby('Department')['Employee'].count()
```

### Common Aggregate Functions:
* `sum()`: Sum of values
* `mean()`: Average of values
* `count()`: Number of non-null observations
* `max()`: Maximum value
* `min()`: Minimum value
* `median()`: Median value
* `std()`: Standard deviation

---

## 3. Grouping by Multiple Columns
You can group data by more than one category by passing a list of column names. This creates a DataFrame with a MultiIndex (hierarchical index).

```python
# Adding a 'Region' column to our dataset for demonstration
df['Region'] = ['North', 'North', 'South', 'South', 'North', 'South']

# Group by both Region and Department
multi_group = df.groupby(['Region', 'Department'])['Salary'].mean()
print(multi_group)
```

---

## 4. The `.agg()` Method (Multiple Aggregations)
Often, you need to calculate different aggregate functions for different columns, or multiple aggregations for the same column simultaneously. The `.agg()` method allows you to pass a dictionary or a list of functions to achieve this.

```python
# Multiple aggregations on the same column
salary_stats = df.groupby('Department')['Salary'].agg(['min', 'max', 'mean'])
print(salary_stats)

# Different aggregations for different columns
complex_agg = df.groupby('Department').agg({
    'Salary': ['sum', 'mean'],
    'Experience_Years': 'max',
    'Employee': 'count'
})
print(complex_agg)
```

---

## 5. Resetting the Index
When you perform a `groupby()`, the column(s) you grouped by automatically become the new index of the resulting DataFrame. To convert this index back into a regular column (which is often necessary before plotting or exporting data), use `reset_index()`.

```python
# Grouping data (Department becomes the index)
grouped_data = df.groupby('Department')['Salary'].sum()

# Resetting the index to make 'Department' a regular column again
final_df = grouped_data.reset_index()
print(final_df)
```
