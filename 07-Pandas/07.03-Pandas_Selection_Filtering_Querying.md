# Pandas: Data Selection, Filtering & Querying

## Overview
Once a dataset is loaded into a Pandas DataFrame, the next crucial step in the Data Analytics Lifecycle (Process phase) is isolating the specific data you need. Pandas offers powerful, flexible methods to slice, dice, and filter data. This module covers selecting subsets of data, applying boolean conditions, and writing efficient queries.

---

## 1. Setup & Sample Data
Before querying, we need a dataset. Let's create a sample DataFrame representing a company's employee records.

```python
import pandas as pd
import numpy as np

# Creating a sample DataFrame
data = {
    'Employee_ID': [101, 102, 103, 104, 105],
    'Name': ['Aarav', 'Sneha', 'Raj', 'Fatima', 'Priya'],
    'Department': ['IT', 'HR', 'IT', 'Finance', 'HR'],
    'Salary': [85000, 75000, 90000, 105000, 72000],
    'Experience_Years': [4, 3, 5, 8, 2]
}

df = pd.DataFrame(data)
df.set_index('Employee_ID', inplace=True) # Setting custom index
print(df)
```

---

## 2. Selecting Columns and Rows (The Basics)

### Selecting Columns
You can extract a single column (returns a `Series`) or multiple columns (returns a subset `DataFrame`).

```python
# 1. Selecting a single column (Bracket notation - Preferred)
names = df['Name']

# 2. Selecting a single column (Dot notation - Fails if column has spaces)
names = df.Name

# 3. Selecting multiple columns (Pass a list of column names)
subset_df = df[['Name', 'Salary', 'Department']]
```

### Selecting Rows via Slicing
Standard Python slicing works on DataFrames to extract sequential rows.

```python
# Select the first three rows
first_three = df[0:3] 
```

---

## 3. Label-based vs. Integer-based Indexing (`loc` and `iloc`)
This is the most standard and robust way to select data in Pandas. 

### `.iloc` (Integer Location)
`iloc` is strictly **integer-position based**. You ask for rows/columns by their numerical index (0 to length-1).

```python
# Syntax: df.iloc[row_indices, column_indices]

# Get the very first row (returns a Series)
first_employee = df.iloc[0]

# Get the first 3 rows and first 2 columns
top_left_data = df.iloc[0:3, 0:2]

# Get specific rows (0th and 3rd) and specific columns (1st and 3rd)
specific_data = df.iloc[[0, 3], [1, 3]]
```

### `.loc` (Label Location)
`loc` is strictly **label-based**. You ask for rows/columns by their actual index names and column header names.

```python
# Syntax: df.loc[row_labels, column_labels]

# Get the data for Employee_ID 103
emp_103 = df.loc[103]

# Get specific columns for a range of index labels 
# Note: In .loc, both the start and the stop slice bounds are INCLUDED!
salary_data = df.loc[101:103, ['Name', 'Salary']]
```

## 4. Filtering Data (Boolean Masking)
Filtering allows you to extract rows that meet specific logical conditions.

### Single Condition
```python
# Step 1: Create a boolean mask (Series of True/False)
is_it_dept = df['Department'] == 'IT'

# Step 2: Apply the mask to the DataFrame
it_employees = df[is_it_dept]

# Shorthand (Most common usage):
high_earners = df[df['Salary'] > 80000]
```

### Multiple Conditions
When combining conditions, you MUST wrap each condition in parentheses `()` and use bitwise operators:
* `&` for AND
* `|` for OR
* `~` for NOT

```python
# AND condition: IT employees earning more than 80,000
senior_it = df[(df['Department'] == 'IT') & (df['Salary'] > 80000)]

# OR condition: Employees in HR OR Finance
hr_or_finance = df[(df['Department'] == 'HR') | (df['Department'] == 'Finance')]

# NOT condition: Employees NOT in the IT department
non_it_emp = df[~(df['Department'] == 'IT')]
```

## 5. Advanced Querying Functions

Pandas provides built-in methods to make filtering cleaner and more readable.

### The `.isin()` Method
Perfect for filtering against a list of values (avoids long OR statements).

```python
# Select employees in specific departments
target_depts = ['HR', 'Finance']
selected_emps = df[df['Department'].isin(target_depts)]
```

### The `.between()` Method
Great for filtering numeric or date ranges.

```python
# Select employees with salaries between 70,000 and 90,000 (inclusive)
mid_earners = df[df['Salary'].between(70000, 90000)]
```

### Handling Missing Data (`isna` / `notna`)
```python
# Assuming we had missing values, we can filter them out
# returns rows where Salary is NOT missing
valid_salary_df = df[df['Salary'].notna()] 

# returns rows where Salary IS missing
missing_salary_df = df[df['Salary'].isna()] 
```

---

## 6. The `.query()` Method
For complex filtering, the `.query()` method allows you to write conditions as clean, readable strings (very similar to SQL `WHERE` clauses). It is highly optimized for performance on large datasets.

```python
# Filter using standard comparison
rich_it = df.query("Department == 'IT' and Salary >= 85000")

# Referencing external variables using the @ symbol
target_experience = 4
experienced_emps = df.query("Experience_Years > @target_experience")

# Querying with list membership
hr_finance_query = df.query("Department in ['HR', 'Finance']")
```
