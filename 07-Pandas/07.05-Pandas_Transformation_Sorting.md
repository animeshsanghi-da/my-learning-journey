# Pandas: Transformation, Sorting & Ranking

## Overview
Data transformation is a core component of the "Process" and "Analyze" phases in the data analytics lifecycle. Raw data is rarely in the perfect shape or order for analysis. This module covers how to reorder your data, rank variables, apply custom mathematical or text-based transformations, and convert continuous data into categorical bins using the **Pandas** library.

---

## 1. Sorting Data
Sorting helps in identifying trends, top performers, or chronological order. Pandas provides intuitive methods to sort by values or index.

### Sorting by Values
Use `sort_values()` to sort a DataFrame or Series by one or more columns.

```python
import pandas as pd

data = {
    'Name': ['Aarav', 'Sneha', 'Raj', 'Fatima'],
    'Age': [25, 28, 22, 30],
    'Salary': [60000, 75000, 50000, 80000]
}
df = pd.DataFrame(data)

# Sort by a single column (Ascending by default)
df_sorted_age = df.sort_values(by='Age')

# Sort by a single column (Descending)
df_highest_salary = df.sort_values(by='Salary', ascending=False)

# Sort by multiple columns
# Sorts by 'Age' first, then 'Salary' descending if ages are tied
df_multi_sort = df.sort_values(by=['Age', 'Salary'], ascending=[True, False])
```

### Sorting by Index
Sometimes, after filtering or grouping, the index becomes jumbled. You can restore order using `sort_index()`.

```python
# Sort rows by index labels
df_sorted_idx = df.sort_index(ascending=True)

# Sort columns alphabetically
df_sorted_cols = df.sort_index(axis=1)
```

## 2. Ranking Data
Ranking is useful for creating leaderboards or percentile analyses. The `rank()` method computes numerical data ranks along an axis.

```python
# Rank salaries from highest (1) to lowest
df['Salary_Rank'] = df['Salary'].rank(ascending=False, method='dense')

# Methods for handling ties:
# 'average' (default): average rank of the tied group
# 'min': lowest rank in the group
# 'max': highest rank in the group
# 'dense': like 'min', but rank always increases by 1 between groups
```

## 3. Data Transformation Methods
Transformation involves modifying values within a column. Pandas provides three primary methods: `map()`, `apply()`, and `applymap()`.

### `map()` - Series Only
Best used for substituting each value in a Series with another value using a dictionary, function, or another Series.

```python
# Map gender categories to binary values
gender_map = {'Male': 0, 'Female': 1}

# Assuming we have a 'Gender' column
# df['Gender_Code'] = df['Gender'].map(gender_map)
```

### `apply()` - Series & DataFrames
Used to apply a custom function (often a `lambda` function) along an axis of the DataFrame or on a Series.

```python
# Apply a lambda function to calculate a 10% bonus on Salary
df['Bonus'] = df['Salary'].apply(lambda x: x * 0.10)

# Apply a function across multiple columns (row-wise operation, axis=1)
def calculate_total(row):
    return row['Salary'] + row['Bonus']

df['Total_Comp'] = df.apply(calculate_total, axis=1)
```

## 4. String Vectorization (Text Transformation)
When cleaning text data, Pandas provides the `.str` accessor, which allows you to apply Python string methods to an entire column at once, efficiently handling missing values (NaN).

```python
# Convert all names to UPPERCASE
df['Name'] = df['Name'].str.upper()

# Check if a string contains a specific substring
df['Is_A_Name'] = df['Name'].str.contains('A')

# Replace characters
# Example: removing currency symbols if they were strings
# df['Revenue'] = df['Revenue'].str.replace('$', '').astype(float)

# String length
df['Name_Length'] = df['Name'].str.len()
```

## 5. Binning and Discretization
Binning is the process of converting continuous, numerical variables into discrete, categorical bins. This is highly useful for creating histograms or grouping data (e.g., converting "Age" into "Age Groups").

### `pd.cut()` - Equal-Width Bins / Custom Boundaries
```python
# Define custom bin edges
bins = [20, 25, 30, 100]
labels = ['Young Adult', 'Adult', 'Senior']

# Segment the 'Age' column into the defined bins
df['Age_Group'] = pd.cut(df['Age'], bins=bins, labels=labels)
```

### `pd.qcut()` - Equal-Frequency Bins (Quantiles)
```python
# Divide the data into 4 bins (quartiles), ensuring an equal number of records in each bin
df['Salary_Quartile'] = pd.qcut(df['Salary'], q=4, labels=['Q1', 'Q2', 'Q3', 'Q4'])
```
