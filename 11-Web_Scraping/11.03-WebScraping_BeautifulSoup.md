# Parsing HTML with BeautifulSoup

## Overview
Once you have fetched a webpage's content using the `requests` module, the next step is to parse that raw HTML into a structured format that Python can understand. **BeautifulSoup** is the industry-standard library for this task, allowing you to navigate the "DOM" (Document Object Model) and extract specific data points like titles, prices, or links.

---

## 1. Setting Up BeautifulSoup
To use BeautifulSoup, you first need to install it along with a parser (like `lxml` or the built-in `html.parser`).

```python
# Installation
# pip install beautifulsoup4 lxml

from bs4 import BeautifulSoup
import requests

url = "https://example.com"
response = requests.get(url)
html_content = response.text

# Initialize the "soup" object
soup = BeautifulSoup(html_content, "lxml")
```

---

## 2. Navigating the HTML Tree
BeautifulSoup allows you to access elements directly by their tag names.

* **Get Page Title:** `print(soup.title.string)` 
* **Get First Paragraph:** `print(soup.p)` 
* **Get Parent Element:** `print(soup.p.parent.name)` 

---

## 3. Searching the DOM
While direct navigation is fast, most web scraping involves searching for specific tags, classes, or IDs.

### find() vs find_all()
* **find():** Returns the *first* occurrence of an element.
* **find_all():** Returns a *list* of all occurrences of an element.

```python
# Find the first h1 tag
header = soup.find("h1")

# Find all hyperlinks on the page
links = soup.find_all("a")
for link in links:
    print(link.get("href"))
```

### Searching by Attributes (Class and ID)
Since `class` is a reserved keyword in Python, BeautifulSoup uses `class_` to search for CSS classes.

```python
# Find a specific div by its ID
main_content = soup.find("div", id="main-container")

# Find all items with a specific class
prices = soup.find_all("span", class_="price-tag")
for price in prices:
    print(price.text.strip())
```

---

## 4. CSS Selectors with select()
For complex queries, you can use `.select()`, which supports standard CSS selector syntax (the same ones used in Matplotlib/Seaborn or web development).

```python
# Selecting by class (.)
items = soup.select(".product-list-item")

# Selecting by ID (#)
footer = soup.select("#footer-section")

# Nested selection (all <a> tags inside a <div> with class 'nav')
nav_links = soup.select("div.nav > a")
```

---

## 5. Extracting Data
After finding an element, you usually want to extract its text or its attributes (like a URL).

* **.text / .get_text():** Returns the text content inside the tag, excluding the HTML tags.
* **.get('attribute'):** Extracts the value of a specific attribute (e.g., `href`, `src`, `alt`).

```python
element = soup.find("a", class_="source-link")

text_data = element.text.strip()  # "Click Here"
url_data = element.get("href")    # "https://origin-site.com"
```

---

## 6. Practical Workflow: Scraping a List
When scraping real-world data (like the "Coders of Delhi" dataset or Olist E-commerce insights), the workflow generally follows these steps:

1.  **Inspect:** Open the website in a browser, right-click, and select "Inspect" to find the tags and classes containing the data.
2.  **Request:** Use `requests.get()` to pull the HTML.
3.  **Parse:** Create a `BeautifulSoup` object.
4.  **Loop:** Use `find_all()` or `select()` to iterate through rows/items and extract text into a list or a Pandas DataFrame.
