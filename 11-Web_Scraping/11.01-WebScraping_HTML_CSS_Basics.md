# HTML & CSS Basics for Web Scraping

## Overview
Web scraping is the process of automatically extracting data from websites. To do this effectively, an analyst must understand the underlying structure of web pages: **HTML** (HyperText Markup Language) for content and **CSS** (Cascading Style Sheets) for styling.

---

## 1. Understanding HTML Structure
HTML uses a hierarchical "tree" structure made of **tags**. Every element on a web page is wrapped in a tag that tells the browser what kind of content it is.

### Common HTML Tags for Scraping
* **`<html>`**: The root element that wraps the entire page.
* **`<head>`**: Contains metadata (title, scripts, styles) not visible on the page.
* **`<body>`**: Contains all visible content like text, images, and links.
* **`<h1>` to `<h6>`**: Header tags, often used for titles or section names.
* **`<p>`**: Paragraph tags used for blocks of text.
* **`<a>`**: Anchor tags used for hyperlinks (the `href` attribute contains the URL).
* **`<div>`**: A generic container used to group elements together.
* **`<span>`**: An inline container for small pieces of text.
* **`<table>`, `<tr>`, `<td>`**: Used for structured data (Table, Table Row, Table Data).
* **`<ul>`, `<li>`**: Used for unordered lists and list items.

### Tag Anatomy
```html
<a href="https://www.example.com" class="link-style" id="nav-home">Click Here</a>
```
* **Tag Name**: `a`
* **Attribute**: `href` (value: "https://www.example.com")
* **Class**: `link-style` (used to group similar elements)
* **ID**: `nav-home` (a unique identifier for that specific element)
* **Text Content**: "Click Here"

## 2. CSS Selectors: Navigating the Tree
CSS Selectors are the "addresses" used by scraping libraries like BeautifulSoup to find specific data points.

### Essential Selectors
* **Type Selector**: Selects all elements of a specific tag.
  * *Example*: `p` selects every paragraph.
* **ID Selector**: Selects a single unique element using a `#`.
  * *Example*: `#price-tag` selects the element with `id="price-tag"`.
* **Class Selector**: Selects all elements with a specific class using a `.`.
  * *Example*: `.product-title` selects every element with `class="product-title"`.
* **Attribute Selector**: Selects elements based on their attributes.
  * *Example*: `a[href]` selects all links with an href attribute.
* **Descendant Selector**: Selects elements inside other elements.
  * *Example*: `div.container p` selects all paragraphs inside a div with the class "container".

---

## 3. The Inspect Tool (Developer Tools)
The most important tool for a web scraper is the browser's **Inspect** tool.
1. Right-click on any element on a webpage.
2. Select **Inspect** (or press `F12`).
3. This opens the **Elements** tab, showing you the exact HTML structure and CSS classes you need to target in your Python script.

---

## 4. Key Concepts for Scraping
* **The DOM (Document Object Model)**: The browser represents the HTML as a logical tree. Scraping is essentially traversing this tree to reach specific nodes.
* **Nested Elements**: Data is often buried deep. For example, to get a product price, you might need to find a `div`, then a `span`, then the text.
* **Static vs. Dynamic Content**: Some HTML is written directly in the file (Static), while some is loaded later via JavaScript (Dynamic). Basic scraping with `requests` only sees Static content.
