# Spreadsheet Basics: Data Entry, Formatting, and Functions

## Overview
Spreadsheets (such as Microsoft Excel and Google Sheets) are the foundational tools for data analytics. As emphasized in the **Google Data Analytics Professional Certificate**, mastering spreadsheets is essential for the *Prepare*, *Process*, and *Analyze* phases of the data lifecycle. They allow analysts to store, organize, manipulate, and visualize data efficiently.

---

## 1. The Anatomy of a Spreadsheet
Before manipulating data, it is crucial to understand how spreadsheets are structured:
* **Workbook:** The entire file containing one or more spreadsheets.
* **Worksheet (Sheet):** A single tab within a workbook.
* **Columns:** Vertical ranges of data, labeled with letters (A, B, C...). Usually represent **Attributes** or **Variables**.
* **Rows:** Horizontal ranges of data, labeled with numbers (1, 2, 3...). Usually represent individual **Records** or **Observations**.
* **Cell:** The intersection of a row and a column (e.g., `B4`).

---

## 2. Data Entry & Best Practices
Clean data analysis starts with proper data entry.
* **Atomic Data:** Each cell should contain only one piece of information (e.g., separate "First Name" and "Last Name" into two columns instead of one "Full Name" column).
* **Headers:** Always use the first row for clear, descriptive column headers. Freeze the top row so headers remain visible when scrolling.
* **Avoid Blank Rows/Columns:** Do not use blank rows or columns to format or separate data, as this breaks sorting, filtering, and pivot tables.
* **Data Validation:** Use data validation rules to restrict what users can enter into a cell (e.g., creating drop-down lists for categories to prevent typos).

---

## 3. Data Formatting
Formatting changes how data *looks* without changing its underlying *value*. This is vital for readability and accuracy.

### Number Formatting
* **Currency:** Formats numbers with monetary symbols and decimals (e.g., `₹1,200.50`).
* **Percentages:** Multiplies the underlying number by 100 and adds a `%` sign (e.g., `0.25` becomes `25%`).
* **Dates:** Standardizes date formats (e.g., `MM/DD/YYYY` vs `YYYY-MM-DD`). 

### Conditional Formatting
Conditional formatting automatically applies visual styles (colors, bolding, icons) to cells based on their values.
* **Highlighting Outliers:** e.g., Highlight all cells in a "Revenue" column greater than $10,000 in green.
* **Color Scales:** Applies a gradient (e.g., red to green) across a range to visually map low to high values.
* **Finding Duplicates:** Quickly identify duplicate records in a dataset by formatting duplicate values in red.

## 4. Essential Formulas and Functions
A **Formula** is an expression written by the user that calculates a value (e.g., `=A1+B1`). A **Function** is a pre-defined formula built into the spreadsheet software (e.g., `=SUM(A1:B1)`).

### Basic Math & Aggregation
```excel
=SUM(A1:A10)       // Adds all values in the specified range
=AVERAGE(B1:B10)   // Calculates the mathematical mean
=COUNT(C1:C10)     // Counts the number of cells containing numerical data
=COUNTA(D1:D10)    // Counts the number of non-empty cells (includes text)
=MAX(E1:E10)       // Finds the highest value in the range
=MIN(F1:F10)       // Finds the lowest value in the range
```

### Logical Functions
Logical functions allow you to evaluate conditions and make decisions within the dataset.
```excel
=IF(logical_expression, value_if_true, value_if_false)
// Example: =IF(A2 > 100, "High", "Low")  
// If the value in A2 is greater than 100, return "High", otherwise return "Low".

=COUNTIF(range, criterion)
// Example: =COUNTIF(B2:B20, "Completed")
// Counts how many times the word "Completed" appears in the range.

=SUMIFS(sum_range, criteria_range1, criterion1, [criteria_range2, criterion2, ...])
// Example: =SUMIFS(C2:C100, B2:B100, "Sales", A2:A100, ">2023-01-01")
// Sums values in C if the department in B is "Sales" AND the date in A is after Jan 1, 2023.
```

### Lookup and Reference Functions
Used to merge data from different tables or sheets.
```excel
=VLOOKUP(search_key, range, index, [is_sorted])
// Example: =VLOOKUP("E145", A2:D100, 3, FALSE)
// Searches for employee ID "E145" in the first column of range A2:D100. 
// When found, it returns the value in the 3rd column of that row. FALSE ensures an exact match.

// Note: XLOOKUP (in newer Excel versions) or INDEX/MATCH are often preferred over VLOOKUP for better performance and flexibility.
```

### Text Manipulation Functions (Data Cleaning)
These are heavily used in the "Process" (cleaning) phase.
```excel
=CONCATENATE(A2, " ", B2)  // Combines the text from A2 and B2 with a space in between.
=TRIM(C2)                  // Removes leading, trailing, and repeated spaces from text.
=UPPER(D2)                 // Converts text to completely UPPERCASE.
=PROPER(E2)                // Capitalizes the first letter of each word.
=LEFT(F2, 4)               // Extracts the first 4 characters from the left side of a string.
=LEN(G2)                   // Returns the total number of characters in a string.
```

---

## 5. Data Manipulation Tools
Beyond functions, spreadsheets offer powerful built-in tools for data analysis:
* **Sorting:** Ordering data alphabetically, numerically, or chronologically (A-Z, Z-A). Always ensure you expand the selection to sort entire rows, not just a single column.
* **Filtering:** Temporarily hiding rows that do not meet specific criteria (e.g., viewing only records where `Status = 'Pending'`).
* **Pivot Tables:** The most powerful tool in a spreadsheet. Pivot tables allow you to summarize, group, and aggregate large datasets dynamically without writing complex formulas.
