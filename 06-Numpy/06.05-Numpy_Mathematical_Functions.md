# NumPy: Built-in Mathematical Functions

## Overview
NumPy provides a massive library of high-level mathematical and statistical functions that operate on arrays. These built-in functions execute highly optimized C code under the hood, making them significantly faster than standard Python math operations. 

Here are the most common NumPy methods frequently used for statistical and mathematical operations in Data Science.

---

## Core Statistical & Aggregation Functions

### 1. `np.mean()`
Computes the arithmetic mean (average) of the elements in an array.
```python
import numpy as np
np.mean(arr)
```

### 2. `np.std()`
Computes the standard deviation of an array, which measures the amount of variation or dispersion in the dataset.
```python
np.std(arr)
```

### 3. `np.var()`
Computes the variance of an array (the square of the standard deviation).
```python
np.var(arr)
```

### 4. `np.median()`
Computes the median (middle value) of an array. It is heavily used in EDA to find the central tendency when outliers are present.
```python
np.median(arr)
```

### 5. `np.percentile()`
Computes the nth percentile of the data along the specified axis.
```python
np.percentile(arr, 50)  # Computes the 50th percentile (which is the median)
```

---

## Min, Max & Accumulation Functions

### 6. `np.min()`
Finds and returns the minimum value present in an array.
```python
np.min(arr)
```

### 7. `np.max()`
Finds and returns the maximum value present in an array.
```python
np.max(arr)
```

### 8. `np.argmin()`
Returns the **index** of the minimum value in an array, rather than the value itself.
```python
np.argmin(arr)
```

### 9. `np.argmax()`
Returns the **index** of the maximum value in an array.
```python
np.argmax(arr)
```

### 10. `np.sum()`
Computes the total sum of all elements in an array (can also be applied along specific axes).
```python
np.sum(arr)
```

### 11. `np.prod()`
Computes the product of all elements in an array.
```python
np.prod(arr)
```

### 12. `np.cumsum()`
Computes the cumulative sum of the elements. It returns an array of the same size where each element is the sum of all preceding elements.
```python
np.cumsum(arr)
```

### 13. `np.diff()`
Computes the n-th discrete difference along the given axis (subtracts the previous element from the current element).
```python
np.diff(arr)
```

---

## Advanced Mathematical & Array Operations

### 14. `np.corrcoef()`
Computes the Pearson product-moment correlation coefficient matrix of two arrays. Very useful in Data Science for identifying feature correlations.
```python
np.corrcoef(arr1, arr2)
```

### 15. `np.unique()`
Finds and returns the unique elements of an array in a sorted order.
```python
np.unique(arr)
```

### 16. `np.linspace()`
Creates an array with evenly spaced numbers over a specified interval. Useful for generating coordinates for plotting graphs.
```python
np.linspace(0, 10, 5)  # Generates 5 evenly spaced numbers from 0 to 10
```

### 17. `np.log()`
Computes the natural logarithm (base *e*) of each element in the array element-wise.
```python
np.log(arr)
```

### 18. `np.exp()`
Computes the exponential ($e^x$) of all elements in the input array.
```python
np.exp(arr)
```
