# Strings, Slicing & String Methods

## Overview
In Python, a string is a sequence of characters enclosed in quotes. Strings are **immutable**, meaning once a string is created, its characters cannot be changed or replaced directly in memory (you must create a new string). Mastering string manipulation is a core skill for Data Analysts, especially during the data cleaning and text processing phases.

---

## 1. Creating Strings
Strings can be created using single, double, or triple quotes. 
* **Single or Double quotes:** Used for standard, single-line strings.
* **Triple quotes (`'''` or `"""`):** Used for multi-line strings or docstrings.

```python
name = "Animesh"
role = 'Data Analyst'
description = """This is a multi-line string.
It can span across multiple lines easily."""
```

---

## 2. String Indexing
Every character in a string has an assigned index (position). Python supports both positive and negative indexing.

* **Positive Indexing:** Starts from `0` (left to right).
* **Negative Indexing:** Starts from `-1` (right to left).

| Character          | A   | n   | i   | m   | e   | s   | h   |
| :----------------- | :-- | :-- | :-- | :-- | :-- | :-- | --- |
| **Positive Index** | 0   | 1   | 2   | 3   | 4   | 5   | 6   |
| **Negative Index** | -7  | -6  | -5  | -4  | -3  | -2  | -1  |

```python
word = "Animesh"
print(word[0])   # Output: A
print(word[-1])  # Output: h (last character)
```

### Calculating Negative Indices in Python
You can calculate the equivalent negative index of an element in a sequence by subtracting the total length of the sequence from its positive index.
* ***`Negative Index = Positive Index - Length`**
* *(Example: The negative index of **m** is: `3 - 7 = -4`.)*

## 3. String Slicing
Slicing allows you to extract a substring from a string.
**Syntax:** `string[start : stop : step]`
* `start`: The starting index (inclusive). Defaults to `0`.
* `stop`: The ending index (exclusive). Defaults to the end of the string.
* `step`: The increment value. Defaults to `1`.

```python
text = "DataAnalytics"

# Basic Slicing
print(text[0:4])    # Output: Data (characters from index 0 to 3)
print(text[4:])     # Output: Analytics (from index 4 to end)
print(text[:4])     # Output: Data (from start to index 3)

# Negative Slicing
print(text[-9:-1])  # Output: Analytic
print(text[:-1])    # Output: DataAnalytic (everything except the last character)

# Slicing with Step
print(text[0:10:2]) # Output: DtAay (Every 2nd character)
print(text[::-1])   # Output: scitylanAataD (Reverses the string completely!)
```

## 4. Essential String Methods
Because strings are objects in Python, they come with built-in methods (functions attached to the object). Since strings are immutable, these methods return a *new* string rather than modifying the original one.

### Case Manipulation
```python
s = "thIS is It"

# Case Manipulation
print(s.upper())       # "THIS IS IT"
print(s.lower())       # "this is it"
print(s.title())       # "This Is It" (Capitalizes first letter of every word)
print(s.capitalize())  # "This is it" (Capitalizes first letter only, makes the rest lowercase)

# These are rarely used, but useful.
print(s.swapcase())    # "THis IS iT" (Inverts the case of every character)
print(s.casefold())    # "this is it" (Similar to lower(), but more aggressive for Unicode)

# Case Checking (Boolean)
# These methods return True or False based on the current case of the string:
print(s.islower())     # False (Checks if all cased characters are lowercase)
print(s.isupper())     # False (Checks if all cased characters are uppercase)
print(s.istitle())     # False (Checks if each word starts with an uppercase letter)
```

### Searching & Counting
```python
s = "Data Science and Data Analytics"

# Counting
print(s.count("Data"))      # 2 (Counts occurrences of the substring)
print(len(s))   # FUNCTION  # 31 (Returns the total number of characters of the string)

# Searching
print(s.find("Science"))    # 5 (Returns the lowest starting index of the substring. Returns -1 if not found)
print(s.rindex("Data"))     # 17 (Returns the index of the last occurrence)
print(s.rfind("Data"))      # 17 (Returns the index of the last occurrence)

# find() is used over index(), because of its error handling.
print(s.find("science"))    # -1 (Returns -1 instead of raising an error)
print(s.index("science"))   # (Would raise a ValueError if the substring is not found)

# Substring Matching & Membership (Boolean)
print(s.endswith("cs"))     # True (Checks if string ends with "cs")
print(s.startswith("Data")) # True (Checks if string starts with "Data")
print("Data" in s) # OPERATOR # True (Checks if a substring exists within a string)
```

### Cleaning & Replacing
These methods are highly utilized in Data Cleaning pipelines (e.g., Pandas).
```python
dirty_text = "   hello world!   "
product_code = "000049102"
price = "$100.50"
emp_id = "EMP8842EMP"

# Cleaning
print(dirty_text.strip())        # "hello world!" (Removes leading and trailing whitespaces)
print(dirty_text.lstrip())       # "hello world!   " (Removes left/leading spaces only)
print(dirty_text.rstrip())       # "   hello world!" (Removes right/trailing spaces only)
print(product_code.lstrip("0"))  # "49102" (Removes left/leading zeros only)

# New methods introduced in Python 3.9
print(emp_id.removeprefix("EMP"))# "8842EMP" (Targets only the exact starting characters)
print(emp_id.removesuffix("EMP"))# "EMP8842" (Targets only the exact ending characters)

# Replacing
print(price.replace("$", ""))    # "100.50" (Replaces all occurrences of a substring)
```
Some Advanced Techniques
```python
# translate() is heavily optimized in C and is the fastest way to strip 
# all punctuation from a massive text dataset in one pass.
import string
review = "Wow!!! This product is amazing, don't you think?"
table = str.maketrans('', '', string.punctuation) # Maps punctuation to 'None'
print(review.translate(table))   # "Wow This product is amazing dont you think"

# strip() only cleans the ends of a string, not the middle. By splitting 
# the string into a list of words (which automatically ignores all extra 
# whitespace) and joining them back with a single space, you normalize the text.
address = "123      Main \n  St."
print(" ".join(address.split())) # "123 Main St."
```

### Splitting & Joining
```python
# Splitting a string into a list
csv_row = "Animesh,Data Analyst,Python"
parsed_row = csv_row.split(",")  
print(parsed_row)  # ['Animesh', 'Data Analyst', 'Python']

# Joining a list into a string
words = ['Data', 'Science', 'Rocks']
sentence = " ".join(words)
print(sentence)    # "Data Science Rocks"
```

## 5. String Formatting (f-strings)
Introduced in Python 3.6, **f-strings** (formatted string literals) provide a concise and convenient way to embed expressions inside string literals for formatting.

```python
name = "Animesh"
role = "Data Analyst"
experience = 2

# Using f-strings (Recommended)
profile = f"My name is {name}. I am a {role} with {experience} years of experience."
print(profile)

# Mathematical operations inside f-strings
print(f"Next year, I will have {experience + 1} years of experience.")
```

## 6. Escape Sequences
Escape sequences allow you to include special characters inside a string that would otherwise cause a syntax error or be interpreted differently.

* `\n`: Newline
* `\t`: Tab
* `\'`: Single Quote
* `\"`: Double Quote
* `\\`: Backslash

```python
print("Hello\nWorld") 
# Output:
# Hello
# World

print("Item:\tApple") 
# Output: Item:   Apple
```
