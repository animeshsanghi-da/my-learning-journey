# Lists and Tuples: Ordered Data Structures in Python

## Overview
In Python, **Lists** and **Tuples** are built-in data structures used to store collections of items. While they share many similarities—such as being ordered and allowing duplicate values—they have one major difference: **Lists are mutable** (can be changed), whereas **Tuples are immutable** (cannot be changed after creation).

---

## 1. Python Lists
A list is a container used to store a sequence of various data types. Lists are defined by enclosing elements in square brackets `[]`.

### Creating and Indexing Lists
Lists maintain the order of elements, and each element can be accessed using its index (starting from `0`).

```python
# Creating an empty list
empty_list = []

# Creating a list of a single value
fruit_list = ["Apple"]

# Creating a list of various data types
mixed_list = ["Apple", 3.14, 42, True]

# Accessing elements via indexing (using the mixed_list)
print(mixed_list[0])   # Output: Apple
print(mixed_list[2])   # Output: 42

# Negative indexing starts from the end
print(mixed_list[-1])  # Output: True
```

### List Slicing
You can extract a sub-list (a slice) using the syntax `list[start:stop:step]`.

```python
friends = ["Animesh", "Rohan", "Sam", "Akash", "Divya"]

print(friends[0:3])    # Output: ['Animesh', 'Rohan', 'Sam'] (index 3 is excluded)
print(friends[1:])     # Output: ['Rohan', 'Sam', 'Akash', 'Divya']
print(friends[::-1])   # Output: ['Divya', 'Akash', 'Sam', 'Rohan', 'Animesh'] (Reverses the list)
```

### List Methods
Because lists are mutable, Python provides numerous built-in methods to modify them in place.

```python
numbers = [1, 8, 7]

# ADDING ELEMENTS

# 1. append(value): Adds an element to the end of the list
numbers.append(45)              # List becomes: [1, 8, 7, 45]

# 2. extend(iterable): Adds all elements from an iterable to the end.
more_numbers = [2, 21, 15]
numbers.extend(more_numbers)    # List becomes: [1, 8, 7, 45, 2, 21, 15]

# 3. insert(index, value): Inserts a value at the specified index
numbers.insert(2, 333)          # List becomes: [1, 8, 333, 7, 45, 2, 21, 15]


# REMOVING

# 4. remove(value): Removes the first occurrence of the specified value
numbers.remove(21)              # List becomes: [1, 8, 333, 7, 45, 2, 15]

# 5. pop(index): Removes and returns the element at the given index. 
# If no index is provided, it removes the last element.
popped_val = numbers.pop(2)     # Returns: 333. List becomes: [1, 8, 7, 45, 2, 15]

# 6. clear(): Removes ALL elements from the list. (Takes no arguments)
numbers.clear()                 # List becomes: []
numbers = [1, 8, 7, 2, 21, 15, 45]


# MANAGING

# 7. sort(): Sorts the list in ascending order
numbers.sort()                  # List becomes: [1, 2, 7, 8, 15, 21, 45]

# 8. reverse(): Reverses the elements of the list
numbers.reverse()               # List becomes: [45, 21, 15, 8, 7, 2, 1]

# 9. copy(): Returns a shallow copy of the list. (Takes no arguments)
new_numbers = numbers.copy()    # new_numbers is now: [45, 21, 15, 8, 7, 2, 1]


# INDEX, COUNT, and LEN

# 10. index(value): Returns the index of the first matched item
index_of_seven = numbers.index(7)        # Returns: 4 (Because 7 is at the 4th index in our reversed list)

# 11. count(value): Returns the number of times the specified value appears
count_of_fifteen = numbers.count(15)     # Returns: 1
```

## 2. Python Tuples
A tuple is an immutable data type in Python. Once a tuple is defined, you cannot alter, add, or remove items from it. Tuples are defined using parentheses `()`.

### Creating Tuples
```python
# Creating an empty tuple
empty_tuple = ()

# Creating a tuple with multiple elements
my_tuple = (1, 7, 2)

# IMPORTANT: Creating a tuple with a single element requires a trailing comma
single_tuple = (1,)  # Correct
not_a_tuple = (1)    # Incorrect, Python treats this as an integer
```

### Why use Tuples?
* **Safety:** If you have data that should *never* change during the execution of your program (e.g., days of the week, fixed coordinates), use a tuple.
* **Performance:** Tuples are slightly faster and more memory-efficient than lists.

### Tuple Methods
Since tuples cannot be modified, they only have two standard built-in methods:

```python
t = (1, 2, 4, 2, 3, 2, 5)

# 1. count(value): Returns the number of times a value occurs in the tuple
print(t.count(2))  # Output: 3

# 2. index(value): Returns the index of the first occurrence of the specified value
print(t.index(4))  # Output: 2
```

---

## 3. Lists vs. Tuples Summary

| Feature | Lists | Tuples |
| :--- | :--- | :--- |
| **Syntax** | `[]` | `()` |
| **Mutability** | Mutable (can be changed) | Immutable (cannot be changed) |
| **Memory** | Consumes more memory | More memory efficient |
| **Methods** | Many (`append`, `remove`, etc.) | Few (`count`, `index`) |
| **Use Case** | Dynamic collections of items | Fixed, read-only collections |
