# Loops and Control Statements in Python

## Overview
In programming, we often need to execute a block of code multiple times. Writing the same code over and over violates the DRY (Don't Repeat Yourself) principle. Python provides loops to handle repetitive tasks efficiently. 

There are two primary types of loops in Python: `while` loops and `for` loops. To manage the flow within these loops, Python provides control statements: `break`, `continue`, and `pass`.

---

## 1. The `while` Loop
A `while` loop executes a block of code as long as a specified condition remains `True`.

### Syntax & Behavior
The condition is evaluated before executing the block. If it is `True`, the body of the loop executes. This process repeats until the condition evaluates to `False`.

```python
i = 1
while i <= 5:
    print("Current value of i is:", i)
    i += 1  # Crucial: Update the loop variable to avoid an infinite loop
```

**Warning on Infinite Loops:** If the condition never becomes `False` (e.g., you forget to increment `i`), the loop will run forever until the program crashes or is manually interrupted.

---

## 2. The `for` Loop
A `for` loop in Python is used to iterate over a sequence (such as a list, tuple, dictionary, set, or string). It works more like an iterator method found in other object-oriented programming languages.

### Iterating through a List
```python
fruits = ["apple", "banana", "cherry", "mango"]

for fruit in fruits:
    print("I like", fruit)
```

### Iterating through a String
```python
word = "PYTHON"
for letter in word:
    print(letter)
```

---

## 3. The `range()` Function
The `range()` function returns a sequence of numbers, starting from 0 by default, and increments by 1 (by default), and stops before a specified number. It is heavily used with `for` loops.

**Syntax:** `range(start, stop, step)`

```python
# Example 1: Stop only (0 to 4)
for i in range(5):
    print(i)

# Example 2: Start and Stop (1 to 5)
for i in range(1, 6):
    print(i)

# Example 3: Start, Stop, and Step (Odd numbers from 1 to 9)
for i in range(1, 10, 2):
    print(i)
```

---

## 4. The `else` Clause in Loops
Python has a unique feature where you can append an `else` block to a `for` or `while` loop. 
The `else` block executes **only if the loop completes its iterations naturally** (i.e., without being stopped by a `break` statement).

```python
for i in range(3):
    print(i)
else:
    print("Loop finished successfully!")
```

---

## 5. Loop Control Statements
Control statements change the execution from its normal sequence. 

### A. The `break` Statement
The `break` statement terminates the loop entirely. Execution proceeds to the first statement immediately following the loop.

```python
for i in range(1, 10):
    if i == 5:
        print("Breaking the loop at i = 5")
        break  # The loop stops here
    print(i)
```

*(Note: If a loop is terminated by `break`, its corresponding `else` block will **not** execute.)*

### B. The `continue` Statement
The `continue` statement skips the rest of the code inside the loop for the current iteration only. The loop does not terminate; it simply jumps to the next iteration.

```python
for i in range(1, 6):
    if i == 3:
        print("Skipping 3")
        continue  # Skips the print(i) for 3
    print("Number:", i)
```

### C. The `pass` Statement
`for` loops, `while` loops, `if` statements, or functions cannot be completely empty in Python, or an `IndentationError` will occur. 
The `pass` statement is a null operation; it does nothing. It is used as a placeholder when a statement is required syntactically but you do not want any command or code to execute.

```python
for i in range(5):
    # I will write this logic later
    pass

print("Program continues...")
```

### Using continue, break, and pass in while loop using if-elif statement.
```python
i = 0               
while i < 20:
    i += 1          
    
    if i == 5:
        continue    # Skips the print statement and jumps back to the top
    elif i == 10:
        break       # Exits the loop entirely
    elif i == 8:
        pass        # Does nothing, just passes to the next line
        
    print("Current value of i is:", i)
```

---

## 6. Nested Loops
You can put a loop inside another loop. The "inner loop" will be executed one time for each iteration of the "outer loop".

```python
colors = ["red", "green"]
items = ["apple", "car"]

for color in colors:
    for item in items:
        print(color, item)
# Output:
# red apple
# red car
# green apple
# green car
```

```python
fruits = ["apple", "banana"]
# Nested for loop
for fruit in fruits:
    for letter in fruit:
        print(letter, fruit)
```
