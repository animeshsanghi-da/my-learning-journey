# Dictionaries & Sets: Properties & Operations

## Overview
In Python, **Dictionaries** and **Sets** are built-in data structures used to store collections of data. While lists and tuples are indexed by a range of numbers, dictionaries are indexed by "keys", and sets are unindexed collections of unique elements. Understanding these is crucial for data manipulation and fast lookups.

---

## 1. Dictionaries in Python
A dictionary is a collection of key-value pairs. It is used to map or associate things you want to store (values) to keys you need to get them.

### Properties of a Dictionary
1. **Unordered:** The items in a dictionary have no defined order (though from Python 3.7 onwards, they maintain insertion order).
2. **Mutable:** You can change, add, or remove items after the dictionary is created.
3. **Indexed:** Values are accessed using their corresponding keys.
4. **No Duplicate Keys:** A dictionary cannot have two items with the same key. If a duplicate key is assigned, the last assigned value overwrites the previous one.

### Creating and Accessing Dictionaries
```python
a = {}         # Creating an empty dictionary.
b = {"a": 20}  # Creating a dictionary with a single key-value pair.

# Creating a dictionary with multiple key-value pairs.
student = {
    "name": "Animesh",
    "course": "Data Analytics",
    "marks": [92, 85, 88],
    "is_certified": True
}

# Accessing elements
print(student["name"])      # Output: Animesh
print(student["marks"][0])  # Output: 92

# Adding or changing - values and keys
student["age"] = 29              # Adds a new key-value pair
student["course"] = "Data Sci"   # Updates the value of existing key

# Single key renaming
student["first_name"] = student.pop("name")

# Bulk key Renaming
mapping = {"first_name": "NAME", "course": "COURSE", "marks": "MARKS"}
for old_key, new_key in mapping.items():
    if old_key in student:
        student[new_key] = student.pop(old_key)
```

## 2. Important Dictionary Methods
Here are the most frequently used methods for dictionary manipulation:

```python
my_dict = {"a": 1, "b": 2, "c": 3}

# 1. items(): Returns a view object of (key, value) tuples
print(my_dict.items())  # dict_items([('a', 1), ('b', 2), ('c', 3)])

# 2. keys(): Returns a view object of all keys
print(my_dict.keys())   # dict_keys(['a', 'b', 'c'])

# 3. values(): Returns a view object of all values
print(my_dict.values()) # dict_values([1, 2, 3])

# 4. update(): Updates the dictionary with supplied key-value pairs
my_dict.update({"c": 4, "d": 5}) 
print(my_dict)          # {'a': 1, 'b': 2, 'c': 4, 'd': 5}

# 5. setdefault(): Adds a key only if it does not already exist. If the key is already there, it returns the existing value and does nothing.
my_dict.setdefault("d", 6)
print(my_dict)          # {'a': 1, 'b': 2, 'c': 4, 'd': 5}

# 6. get(): Returns the value of the specified key
print(my_dict.get("a")) # Output: 1
print(my_dict.get("z")) # Output: None (Does not throw an error)
# print(my_dict["z"])   # This would throw a KeyError!

# 7. pop(): Removes the element with the specified key and returns its value
val = my_dict.pop("b")
print(val)              # Output: 2

# 8. popitem(): Removes and returns the last inserted key-value pair as a tuple
val = my_dict.popitem()
print(val)              # Output: ('d', 5)

# 9. clear(): Removes all elements from the dictionary, leaving it empty
my_dict.clear()
print(my_dict)          # Output: {}
```

```python
# 10. fromkeys(): 
# Creates a new dictionary from an iterable (like a list or tuple) of keys, optionally assigning a shared value to all of them.

# If no values are assigned it sets to default value: None
keys = ('a', 'b', 'c')
dict1 = dict.fromkeys(keys)
print(dict1)      # Output: {'a': None, 'b': None, 'c': None}

keys = ['name', 'age', 'role']
default_value = 'unknown'
user_dict = dict.fromkeys(keys, default_value)
print(user_dict)  # Output: {'name': 'unknown', 'age': 'unknown', 'role': 'unknown'}
```

## 3. Sets in Python
A Set is a collection of non-repetitive (unique) elements. Sets are written with curly brackets `{}`, similar to dictionaries, but without key-value pairs.

### Properties of a Set
1. **Unordered:** Sets are unindexed, meaning you cannot access elements using `[0]` or `[1]`.
2. **Unchangeable (Immutable elements):** You cannot change the items in a set once created, though you can *add* or *remove* items from the set itself.
3. **No Duplicates Allowed:** Sets cannot contain duplicate values. If you try to add a duplicate, it will simply be ignored.
4. **No Mutable Elements inside:** You cannot put lists or dictionaries inside a set because they are mutable (unhashable).

### Creating Sets
```python
# Creating a basic set
my_set = {1, 2, 3, 4, 5, 1, 2} 
print(my_set)  # Output: {1, 2, 3, 4, 5} (Duplicates are automatically removed)

# Creating an empty set
# Note: empty_set = {} creates an empty DICTIONARY, not a set.
empty_set = set() 
print(type(empty_set))  # Output: <class 'set'>
```

## 4. Set Methods & Mathematical Operations
Sets are highly optimized for mathematical operations like finding intersections, unions, and differences (conceptually similar to Venn Diagrams).

### Standard Methods
```python
s = set()

# 1. add(): Adds an element to the set
s.add(4)
s.add(5)
# s.add([1, 2]) # Throws an error: unhashable type: 'list'

# 2. len(): Returns the number of elements
print(len(s))   # Output: 2

# 3. remove(): Removes an element. Throws KeyError if not found.
s.remove(5)

# 4. pop(): Removes and returns an arbitrary element
val = s.pop()

# 5. clear(): Empties the set
s.clear()
```

```python
# 6. update(iterable): Adds multiple items from another set or any iterable (list, tuple, etc)
set1 = {1, 2}
set2 = {2, 3}
my_list = [3, 4]
my_tuple = (4, 5)
set3 = {5, 6}

# Updating with Another Set: (most commonly used)
set1.update(set2)
print(set1)  # Output: {1, 2, 3}

# Updating with Multiple Iterables: (lists, tuples, strings)
# Here Python treats a string as an iterable of individual characters
set1.update(my_list, my_tuple, "ab")
print(set1)  # Output: {1, 2, 3, 4, 5, 'a', 'b'} (order will vary)

# Shortcut Syntax (|=)
set1 |= set3
print(set1)  # Output: {1, 2, 3, 4, 5, 6, 'b', 'a'} (order will vary)
```

```python
# 7. discard(element): Removes the specified element if present; does not raise an error if it is missing.
# Initializing a set
fruits = {"apple", "banana", "cherry"}

# 1. Removing an existing item
fruits.discard("banana")
print(fruits)  # Output: {'apple', 'cherry'}

# 2. Attempting to remove an item that is NOT in the set
fruits.discard("orange") 
print(fruits)  # Output: {'apple', 'cherry'} (No error is raised)
```

### Mathematical Operations
These operations are incredibly useful for data cleaning and comparison (e.g., finding overlapping users in two different database tables).

![](../Images/venn_diagram_python.png)

| Method                            | Operator | Description                                            |
| --------------------------------- | -------- | ------------------------------------------------------ |
| **`union(*others)`**              | `\|`     | Returns: all unique items present in all sets.         |
| **`intersection(*others)`**       | `&`      | Returns: common items present in all sets.             |
| **`difference(*others)`**         | `-`      | Returns: items in the first set but not in the others. |
| **`symmetric_difference(other)`** | `^`      | Returns: items present in either set, but not both.    |

```python
set_A = {1, 2, 3, 4}
set_B = {3, 4, 5, 6}

# 1. Union (Elements in either A or B)
# Formula: A U B
print(set_A.union(set_B))  
# Output: {1, 2, 3, 4, 5, 6}

# 2. Intersection (Elements common to both A and B)
# Formula: A ∩ B
print(set_A.intersection(set_B))  
# Output: {3, 4}

# 3. Difference (Elements in A but NOT in B)
# Formula: A - B
print(set_A.difference(set_B))  
# Output: {1, 2}

# 4. Symmetric Difference (Elements in either A or B, but NOT both)
# Formula: A Δ B
print(set_A.symmetric_difference(set_B))  
# Output: {1, 2, 5, 6}
```
