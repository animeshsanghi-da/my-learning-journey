# Operators and Conditional Expressions in Python

## Overview
In Python, **Operators** are special symbols or keywords used to perform operations on variables and values. **Conditional Expressions** (control flow statements) allow your program to make decisions and execute different blocks of code based on whether specific conditions are True or False.

---

## 1. Python Operators
Python divides operators into several main categories.

### Arithmetic Operators
Used to perform common mathematical operations.
```python
a = 10
b = 3

print(a + b)   # Addition: 13
print(a - b)   # Subtraction: 7
print(a * b)   # Multiplication: 30
print(a / b)   # Division (Float): 3.333...
print(a // b)  # Floor Division (Integer): 3
print(a % b)   # Modulus (Remainder): 1
print(a ** b)  # Exponentiation (10 to the power of 3): 1000
```

### Assignment Operators
Used to assign values to variables. They can also be combined with arithmetic operators for shorthand updates.
```python
x = 25      # Standard assignment
x += 3      # Equivalent to x = x + 3 (x becomes 28)
x -= 2      # Equivalent to x = x - 2 (x becomes 26)
x *= 2      # Equivalent to x = x * 2 (x becomes 52)
x /= 2      # Equivalent to x = x / 2 (x becomes 26.0)
x %= 7      # Equivalent to x = x % 7 (x becomes 5.0)
x //= 2     # Equivalent to x = x // 2 (x becomes 2.0)
x **= 2     # Equivalent to x = x ** 2 (x becomes 4.0)
```

### Comparison (Relational) Operators
Used to compare two values. They always return a boolean (`True` or `False`).
```python
p = 10
q = 20

print(p == q)  # Equal to: False
print(p != q)  # Not equal to: True
print(p > q)   # Greater than: False
print(p < q)   # Less than: True
print(p >= 10) # Greater than or equal to: True
print(q <= 15) # Less than or equal to: False
# Chaining allowed
print(p < q < 25) # : True
```

### Logical Operators
Used to combine multiple conditional statements.
* `and`: Returns True if both statements are true.
* `or`: Returns True if one of the statements is true.
* `not`: Reverses the result, returns False if the result is true.

```python
condition1 = True
condition2 = False

print(condition1 and condition2) # False
print(condition1 or condition2)  # True
print(not condition1)            # False
```

### Identity Operators
Used to compare the memory locations of two objects, not just their values.
- `is`: Returns True if both variables point to the exact same object. 
- `is not`: Returns True if both variables do NOT point to the exact same object.
```python
value = None
print(value is None)      # Output: True
print(value is not None)  # Output: False
```

### Membership Operators
Used to test if a sequence (like a string, list, or tuple) is presented in an object.
- `in`: Returns True if a sequence with the specified value is present. 
- `not in`: Returns True if a sequence with the specified value is not present.
```python
valid_categories = ['Tech', 'Finance', 'Healthcare'] 
category = 'Finance' 
print(category in valid_categories)     # Output: True
print(category not in valid_categories) # Output: False
```

### The Walrus Operator (`:=`)
Added in Python 3.8. It allows you to assign values to variables as part of a larger expression. Very useful for optimizing while loops or list comprehensions so you don't calculate the same thing twice.
```python
data = [10, 20, 30, 40, 50]

# Calculates length once, assigns it to 'n', and checks condition simultaneously
if (n := len(data)) > 3:
    print(f"Dataset is large enough. Row count: {n}")
```

## 2. Taking User Input
Before evaluating conditions, we often need dynamic data. In Python, the `input()` function reads a line from the console. By default, `input()` always returns a string.

```python
# Taking input and converting (typecasting) it to an integer
age = int(input("Enter your age: "))
print("You are", age, "years old.")
```

## 3. Conditional Expressions (Control Flow)
Conditionals allow a program to branch its execution path. In Python, indentation (whitespace at the beginning of a line) is mandatory to define blocks of code.

### The `if` Statement
Executes a block of code only if the specified condition is `True`.
```python
age = 20
if age >= 18:
    print("You are eligible to vote.") # Notice the indentation
```

### The `if-else` Statement
Provides an alternative block of code that executes if the condition is `False`.
```python
temperature = 30
if temperature > 35:
    print("It's a hot day.")
else:
    print("The weather is pleasant.")
```

### The `if-elif-else` Ladder
Used when checking multiple conditions sequentially. As soon as one condition evaluates to `True`, its corresponding block executes, and the rest of the ladder is bypassed.
```python
marks = 85

if marks >= 90:
    print("Grade: A")
elif marks >= 80:
    print("Grade: B")
elif marks >= 70:
    print("Grade: C")
else:
    print("Grade: F")
```

---
### Nested `if` Statements
You can place `if` statements inside other `if` statements for complex logic.
```python
num = 15

if num >= 0:
    if num == 0:
        print("Zero")
    else:
        print("Positive number")
else:
    print("Negative number")
```

### Truthy and Falsy Values
In Python, you do not always need to explicitly write `== True` or `== False` in your conditions. Python inherently evaluates certain values as `False` (Falsy) and everything else as `True` (Truthy).

**Falsy Values include:**
* `None`
* Zero of any numeric type: `0`, `0.0`
* Empty sequences and collections: `""` (empty string), `[]` (empty list), `{}` (empty dictionary), `()` (empty tuple)

```python
name = ""
# Because 'name' is an empty string, it evaluates to False.
# 'not False' becomes True, so the block executes.
if not name:
    print("Name is empty!")   # Output: Name is empty!
```

```python
items = ["", 0.0, 0, [], {}, ()]
# 1. Checking the container 
if items:
    print("Truthy")      # (Truthy because the list itself holds 6 items)

# 2. Checking the contents
for item in items:
    if item:
        print(f"{item} is Truthy")
    else:
        print(f"{item} is Falsy")     # (All Falsy)
```

### The Ternary Operator (Conditional Expressions)
A one-line shorthand for an `if-else` statement. It is incredibly useful for simple assignments based on a condition, keeping your code clean and concise.
**Syntax:** `[value_if_true] if [condition] else [value_if_false]`

```python
age = 20
# Standard if-else:
# if age >= 18:
#     status = "Adult"
# else:
#     status = "Minor"

# Ternary equivalent:
status = "Adult" if age >= 18 else "Minor"
print(status) # Output: Adult
```

## 4. The `match-case` Statement (Python 3.10+)
Introduced in Python 3.10, `match-case` operates similarly to the `switch-case` statements found in C++ or Java. It provides a clean way to compare a variable against multiple distinct values.

```python
status_code = 404

match status_code:
    case 200:
        print("Success: OK")
    case 400:
        print("Client Error: Bad Request")
    case 404:
        print("Client Error: Not Found")
    case 500:
        print("Server Error: Internal Server Error")
    case _: # The underscore acts as the 'default' or 'else' case
        print("Unknown Status Code")
```
