# Data Visualization with Matplotlib: Standard Plots

## Overview
**Matplotlib** is the foundational plotting library in Python for 2D graphics. It is highly customizable and serves as the engine under the hood for many other visualization libraries (like Pandas plotting and Seaborn). The `matplotlib.pyplot` module provides a state-based interface similar to MATLAB, which is perfect for creating quick, standard visualizations like Line, Bar, Pie, and Scatter plots.

Data visualization is a critical step in the Data Analytics Lifecycle (the "Analyze" and "Share" phases) to discover patterns, identify outliers, and present findings to stakeholders effectively.

---

## 1. Setting Up Matplotlib
Before creating any plots, you need to import the library. The industry standard alias is `plt`.

```python
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

# Optional: Set a clean style for all plots
plt.style.use('ggplot')
```

---

## 2. Line Plots (The Basics)
While not explicitly in the title, the Line Plot is the default behavior of Matplotlib and the best starting point. It is best used for **time-series data** or tracking continuous changes over time.

### When to use:
* Tracking stock prices over a month.
* Monitoring daily temperature changes.

```python
# Data
months = ['Jan', 'Feb', 'Mar', 'Apr', 'May']
revenue = [25000, 32000, 28000, 41000, 39000]

# Creating the plot
plt.plot(months, revenue, marker='o', color='b', linestyle='--', linewidth=2)

# Customizing the plot
plt.title('Monthly Revenue Tracking')
plt.xlabel('Month')
plt.ylabel('Revenue (in USD)')
plt.grid(True)

# Displaying the plot
plt.show()
```

## 3. Bar Plots (`plt.bar`)
Bar plots represent categorical data with rectangular bars with lengths proportional to the values they represent. 

### When to use:
* Comparing quantities of different categories (e.g., Sales by Region).
* Showing rankings (e.g., Top 5 best-selling products).

### Theoretical Concept:
* **Vertical Bar Chart:** Best when categories have short names (e.g., Months, Departments).
* **Horizontal Bar Chart (`plt.barh`):** Best when category labels are long and would overlap on the x-axis.

```python
# Data
categories = ['Electronics', 'Clothing', 'Home', 'Books', 'Toys']
sales = [15000, 8500, 12000, 4000, 9500]

# Creating a vertical bar chart
plt.figure(figsize=(8, 5))  # Set figure size
plt.bar(categories, sales, color='skyblue', edgecolor='black')

# Adding text labels on top of the bars
for i in range(len(categories)):
    plt.text(i, sales[i] + 500, str(sales[i]), ha='center')

plt.title('Sales by Product Category')
plt.xlabel('Category')
plt.ylabel('Total Sales ($)')
plt.show()
```

## 4. Pie Charts (`plt.pie`)
Pie charts show the proportional distribution of items within a single category. The entire pie represents 100%.

### When to use:
* Showing market share.
* Visualizing the breakdown of a budget.

### Best Practices:
* Do not use a pie chart if you have more than 5-6 categories (it becomes unreadable).
* Use the `explode` parameter to highlight a specific slice.
* Use `autopct` to automatically calculate and display percentages.

```python
# Data
labels = ['Rent', 'Food', 'Transport', 'Utilities', 'Entertainment']
expenses = [1200, 400, 150, 200, 300]
colors = ['#ff9999','#66b3ff','#99ff99','#ffcc99', '#c2c2f0']

# Explode the 1st slice (Rent) to make it stand out
explode = (0.1, 0, 0, 0, 0)  

# Creating the pie chart
plt.figure(figsize=(6, 6))
plt.pie(expenses, explode=explode, labels=labels, colors=colors, 
        autopct='%1.1f%%', shadow=True, startangle=140)

# Ensure the pie is drawn as a perfect circle
plt.axis('equal') 
plt.title('Monthly Expense Breakdown')
plt.show()
```

## 5. Scatter Plots (`plt.scatter`)
Scatter plots use Cartesian coordinates to display values for typically two variables from a dataset. 

### When to use:
* Discovering relationships or **correlations** between two numerical variables.
* Identifying outliers or clusters in a dataset.

### Theoretical Concept:
* **Positive Correlation:** As X increases, Y increases.
* **Negative Correlation:** As X increases, Y decreases.
* **No Correlation:** Points are scattered randomly.
* You can add a 3rd dimension to a 2D scatter plot by changing the **size** (`s`) or **color** (`c`) of the dots based on another variable.

```python
# Generating random data for the example
np.random.seed(42)
marketing_spend = np.random.randint(1000, 10000, 50)
new_customers = marketing_spend * np.random.uniform(0.01, 0.05, 50)

# A 3rd variable to determine the size of the bubble
customer_satisfaction = np.random.randint(10, 100, 50)

plt.figure(figsize=(8, 5))
scatter = plt.scatter(marketing_spend, new_customers, 
                      c=customer_satisfaction, cmap='viridis', 
                      s=customer_satisfaction * 5, alpha=0.6, edgecolors='w')

# Adding a colorbar to explain the color mapping
cbar = plt.colorbar(scatter)
cbar.set_label('Customer Satisfaction Score')

plt.title('Marketing Spend vs. New Customers')
plt.xlabel('Marketing Spend ($)')
plt.ylabel('Number of New Customers')
plt.grid(True, alpha=0.3)
plt.show()
```

## 6. General Customization Checklist
Whenever you create a plot in Matplotlib, ensure you tick these boxes for maximum readability:
1. `plt.title()`: Does the plot have a clear, descriptive title?
2. `plt.xlabel()` & `plt.ylabel()`: Are the axes labeled? Do they include units (e.g., "$", "Years", "Kg")?
3. `plt.legend()`: If there are multiple lines or categories, is there a legend explaining what the colors mean?
4. `plt.figure(figsize=(x, y))`: Is the aspect ratio appropriate for the data?
