# Matplotlib: Subplots & Layouts

## Overview
When performing Exploratory Data Analysis (EDA) or building dashboards, plotting a single chart is often insufficient. You frequently need to compare multiple variables, view different distributions side-by-side, or show the relationship between various features simultaneously. **Matplotlib** provides powerful layout managers, primarily through the creation of **Subplots**, to arrange multiple plots within a single image or window.

---

## 1. The Anatomy of a Matplotlib Layout
To master subplots, you must understand Matplotlib's Object-Oriented hierarchy:
* **Figure (`fig`):** The overall window or page that everything is drawn on. It is the top-level container that can hold one or more Axes.
* **Axes (`ax`):** A single plot or chart within the Figure. An Axes object contains the actual data, labels, ticks, and titles. A Figure can have multiple Axes, arranged in a grid.

---

## 2. Creating Subplots using `plt.subplots()`
The most efficient and modern way to create multiple plots is using the `plt.subplots()` function. It creates a Figure and a grid of Axes all at once, returning them as a tuple.

### 1D Grid (1 Row, Multiple Columns)
When you create a 1D grid, the `axes` object is returned as a 1D NumPy array.

```python
import matplotlib.pyplot as plt
import numpy as np

# Sample Data
x = np.linspace(0, 10, 100)
y1 = np.sin(x)
y2 = np.cos(x)

# Create a figure with 1 row and 2 columns
fig, axes = plt.subplots(nrows=1, ncols=2, figsize=(10, 4))

# Plot on the first subplot (index 0)
axes[0].plot(x, y1, color='blue')
axes[0].set_title('Sine Wave')
axes[0].set_xlabel('X-axis')
axes[0].set_ylabel('Amplitude')

# Plot on the second subplot (index 1)
axes[1].plot(x, y2, color='red')
axes[1].set_title('Cosine Wave')
axes[1].set_xlabel('X-axis')

# Automatically adjust spacing to prevent overlapping labels
plt.tight_layout()
plt.show()
```

### 2D Grid (Multiple Rows and Columns)
When creating a 2D grid (e.g., 2x2), the `axes` object is returned as a 2D NumPy array. You access individual plots using matrix indexing `[row, column]`.

```python
# Create a 2x2 grid of subplots
fig, axes = plt.subplots(nrows=2, ncols=2, figsize=(10, 8))

# Top-Left [0, 0]
axes[0, 0].plot(x, y1, 'b-')
axes[0, 0].set_title('Top Left: Sine')

# Top-Right [0, 1]
axes[0, 1].plot(x, y2, 'r--')
axes[0, 1].set_title('Top Right: Cosine')

# Bottom-Left [1, 0]
axes[1, 0].plot(x, -y1, 'g-.')
axes[1, 0].set_title('Bottom Left: Negative Sine')

# Bottom-Right [1, 1]
axes[1, 1].plot(x, -y2, 'm:')
axes[1, 1].set_title('Bottom Right: Negative Cosine')

plt.tight_layout()
plt.show()
```

---

## 3. Layout Optimization Techniques

### `plt.tight_layout()`
When creating multiple subplots, the labels, titles, and tick marks of adjacent plots often overlap. Calling `plt.tight_layout()` right before `plt.show()` automatically adjusts the padding between and around the subplots to fit beautifully within the Figure area.

### Sharing Axes (`sharex`, `sharey`)
If your subplots have the same scale or represent the same units, sharing axes makes the visualization cleaner and easier to compare, as it removes redundant tick labels.

```python
# Create 3 stacked plots that share the same X-axis
fig, axes = plt.subplots(nrows=3, ncols=1, figsize=(8, 6), sharex=True)

axes[0].plot(x, y1, color='blue')
axes[0].set_ylabel('Sine')

axes[1].plot(x, y2, color='red')
axes[1].set_ylabel('Cosine')

axes[2].plot(x, y1 + y2, color='purple')
axes[2].set_ylabel('Sine + Cosine')
axes[2].set_xlabel('Time (s)') # Only the bottom plot needs the X-label now

plt.tight_layout()
plt.show()
```

## 4. Advanced Layouts with `GridSpec` (Optional but Powerful)
While `plt.subplots()` creates evenly sized grids, `GridSpec` allows you to create complex layouts where some plots span multiple rows or columns (e.g., a dashboard layout).

```python
import matplotlib.gridspec as gridspec

fig = plt.figure(figsize=(10, 6))
# Define a 2x2 grid
gs = gridspec.GridSpec(2, 2)

# Ax1 spans the entire first row
ax1 = fig.add_subplot(gs[0, :])
ax1.plot(x, y1)
ax1.set_title('Spans entirely across the top')

# Ax2 and Ax3 share the bottom row
ax2 = fig.add_subplot(gs[1, 0])
ax2.plot(x, y2)
ax2.set_title('Bottom Left')

ax3 = fig.add_subplot(gs[1, 1])
ax3.plot(x, -y2)
ax3.set_title('Bottom Right')

plt.tight_layout()
plt.show()
```
