# Lambda Functions, Map, Filter, & Reduce

## Overview
As you advance in Python, you will often encounter functional programming paradigms. Python provides several built-in tools like `lambda`, `map`, `filter`, and `reduce` that allow you to write cleaner, more concise, and highly efficient code, especially when dealing with iterables (like lists or tuples).

---

## 1. Lambda Functions (Anonymous Functions)
A `lambda` function is a small anonymous function defined without a name. While standard functions are defined using the `def` keyword, anonymous functions are defined using the `lambda` keyword.

**Syntax:**
`lambda arguments: expression`

* They can take any number of arguments but can only have **one** expression.
* The expression is evaluated and returned automatically.

### Basic Examples
```python
# Traditional function
def square(x):
    return x * x

# Equivalent Lambda function
square_lambda = lambda x: x * x
print(square_lambda(5))  # Output: 25

# Lambda with multiple arguments
add_numbers = lambda a, b, c: a + b + c
print(add_numbers(1, 2, 3))  # Output: 6
```

### Common Use Case: Sorting
Lambda functions are highly useful as the `key` argument in sorting functions.
```python
# Sorting a list of tuples by the second element
pairs = [(1, 'one'), (3, 'three'), (2, 'two')]
pairs.sort(key=lambda pair: pair[1])
print(pairs)  # Output: [(1, 'one'), (3, 'three'), (2, 'two')]
```

## 2. The `map()` Function
The `map()` function applies a given function to all the items in an input iterable (list, tuple, etc.) and returns a map object (which is an iterator).

**Syntax:**
`map(function, iterable)`

### Example: Squaring a List of Numbers
Instead of using a `for` loop, you can use `map()` combined with a `lambda` function.
```python
l = [1, 2, 4, 6, 4, 3]

# Using map with lambda
sqList = list(map(lambda x: x * x, l))
print(sqList)  # Output: [1, 4, 16, 36, 16, 9]

# Using map with a predefined function
def cube(x):
    return x * x * x

cubeList = list(map(cube, l))
print(cubeList)  # Output: [1, 8, 64, 216, 64, 27]
```

---

## 3. The `filter()` Function
The `filter()` function creates a list of elements for which a specific function returns `True`. It "filters" out the elements that evaluate to `False`.

**Syntax:**
`filter(function, iterable)`

### Example: Filtering Even Numbers
```python
def isEven(n):
    return n % 2 == 0

l = [1, 2, 4, 6, 4, 3]

# Using filter with a predefined function
evenList = list(filter(isEven, l))
print(evenList)  # Output: [2, 4, 6, 4]

# Using filter with a lambda function (filtering numbers greater than 3)
greater_than_three = list(filter(lambda x: x > 3, l))
print(greater_than_three)  # Output: [4, 6, 4]
```

---

## 4. The `reduce()` Function
The `reduce()` function applies a rolling computation to sequential pairs of values in an iterable. Unlike `map()` and `filter()`, `reduce()` is not a built-in function in Python 3; it must be imported from the `functools` module.

**Syntax:**
`from functools import reduce`
`reduce(function, iterable)`

### How it Works:
If you have a list `[1, 2, 3, 4]` and apply a `reduce` function that adds two elements, it will compute:
1. `1 + 2 = 3`
2. `3 + 3 = 6`
3. `6 + 4 = 10`
Result: `10`

### Example: Summing a List
```python
from functools import reduce

numbers = [1, 2, 3, 4, 5]

# Sum all numbers in the list
sum_all = reduce(lambda a, b: a + b, numbers)
print(sum_all)  # Output: 15

# Find the maximum number in a list using reduce
max_num = reduce(lambda a, b: a if a > b else b, numbers)
print(max_num)  # Output: 5
```

## Summary Comparison

| Function | Purpose | Returns |
| :--- | :--- | :--- |
| **Lambda** | Creates quick, inline, anonymous functions. | A function object. |
| **Map** | Modifies/transforms every item in an iterable. | An iterable of the exact same length. |
| **Filter** | Keeps only the items that satisfy a `True`/`False` condition. | An iterable containing fewer or equal items. |
| **Reduce** | Rolls up an iterable into a single cumulative value. | A single computed value (int, string, etc.). |

These functional programming tools are critical for writing clean, Pythonic data pipelines, which is a core skill for any Data Analyst utilizing Python.
